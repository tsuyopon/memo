# 概要
MySQLの公式サンプルとして提供しているworld_xについて

# 導入
たったこれだけで利用できるようになります。
```
$ wget downloads.mysql.com/docs/world_x-db.tar.gz
$ tar zxvf world_x-db.tar.gz 
$ cd world_x-db/
$ mysql -u root -p
mysql> SOURCE world_x.sql    // 自分が試した感じいくつかエラーが出たがとりあえず気にせず進める
mysql> use world_x
```

データ情報を覗いてみる。多少量があるのでLIMITなどで情報を限定して取得する。

テーブルスキーマも参照してみる
```
mysql> SHOW CREATE TABLE country;

| Table   | Create Table                                                                                                                                                                                                                               |

mysql> SHOW CREATE TABLE countrylanguage;
+-----------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Table           | Create Table                                                                                                                                                                                                                                                                                                                                                                                                                                      |

| countrylanguage | CREATE TABLE `countrylanguage` (
```
