### 文字変換
char文字変換
```
mysql> select char(49);
+----------+
| char(49) |
+----------+
| 1        |
+----------+
```

ASCII変換
```
mysql> select ascii('1');
+------------+
| ascii('1') |
+------------+
|         49 |
+------------+
```

大文字、小文字への変換
```
mysql> select upper('abc012');
+-----------------+
| upper('abc012') |
+-----------------+
| ABC012          |
+-----------------+

mysql> select lower('ABC012');
+-----------------+
| lower('ABC012') |
+-----------------+
| abc012          |
+-----------------+
```

前後の空白を除去する
```
mysql> SELECT TRIM('          abc          ');
+---------------------------------+
| TRIM('          abc          ') |
+---------------------------------+
| abc                             |
+---------------------------------+
```

左側・右側だけ除去するltrim, rtrimも存在します。
```
mysql> SELECT LENGTH(LTRIM('  abc '));
+-------------------------+
| LENGTH(LTRIM('  abc ')) |
+-------------------------+
|                       4 |
+-------------------------+

mysql> SELECT LENGTH(RTRIM('  abc '));
+-------------------------+
| LENGTH(RTRIM('  abc ')) |
+-------------------------+
|                       5 |
+-------------------------+
```

特定の文字列を置き換える
```
mysql> SELECT REPLACE('a0a1a2', 'a0', 'b1');
+-------------------------------+
| REPLACE('a0a1a2', 'a0', 'b1') |
+-------------------------------+
| b1a1a2                        |
+-------------------------------+
```

特定の位置から指定した文字数を取得する
```
mysql> SELECT SUBSTRING('abc012', 2, 3);
+---------------------------+
| SUBSTRING('abc012', 2, 3) |
+---------------------------+
| bc0                       |
+---------------------------+
```

指定した文字の位置を検索する
```
mysql> select locate('c01', 'abc012');
+-------------------------+
| locate('c01', 'abc012') |
+-------------------------+
|                       3 |
+-------------------------+
```

文字の長さを取得する
```
mysql> select length('abcdefg');
+-------------------+
| length('abcdefg') |
+-------------------+
|                 7 |
+-------------------+
```

特定の文字列を指定された数だけ繰り返す
```
mysql> SELECT REPEAT('abcdefg', 2);
+----------------------+
| REPEAT('abcdefg', 2) |
+----------------------+
| abcdefgabcdefg       |
+----------------------+
```

右文字を詰める
```
mysql> SELECT RPAD('abc', 5, '012');
+-----------------------+
| RPAD('abc', 5, '012') |
+-----------------------+
| abc01                 |
+-----------------------+

mysql> SELECT RPAD('abc', 10, '012');
+------------------------+
| RPAD('abc', 10, '012') |
+------------------------+
| abc0120120             |
+------------------------+
```

左文字を詰める
```
mysql> SELECT LPAD('abc', 5, '012');
+-----------------------+
| LPAD('abc', 5, '012') |
+-----------------------+
| 01abc                 |
+-----------------------+

mysql> SELECT LPAD('abc', 10, '012');
+------------------------+
| LPAD('abc', 10, '012') |
+------------------------+
| 0120120abc             |
+------------------------+
```

### 時刻
```
mysql> SELECT NOW()+0;
+----------------+
| NOW()+0        |
+----------------+
| 20170531070950 |
+----------------+

mysql> SELECT NOW(), CURRENT_TIMESTAMP(), CURRENT_TIMESTAMP;
+---------------------+---------------------+---------------------+
| NOW()               | CURRENT_TIMESTAMP() | CURRENT_TIMESTAMP   |
+---------------------+---------------------+---------------------+
| 2016-12-22 23:36:05 | 2016-12-22 23:36:05 | 2016-12-22 23:36:05 |
+---------------------+---------------------+---------------------+

mysql> SELECT CURDATE(), CURRENT_DATE(), CURRENT_DATE;
+------------+----------------+--------------+
| CURDATE()  | CURRENT_DATE() | CURRENT_DATE |
+------------+----------------+--------------+
| 2016-12-22 | 2016-12-22     | 2016-12-22   |
+------------+----------------+--------------+

mysql> SELECT CURTIME(), CURRENT_TIME(), CURRENT_TIME;
+-----------+----------------+--------------+
| CURTIME() | CURRENT_TIME() | CURRENT_TIME |
+-----------+----------------+--------------+
| 23:36:41  | 23:36:41       | 23:36:41     |
+-----------+----------------+--------------+
```

フォーマットを指定する。
```
mysql> select date_format(now(), '%Y.%m.%d');
+--------------------------------+
| date_format(now(), '%Y.%m.%d') |
+--------------------------------+
| 2017.05.30                     |
+--------------------------------+
```

日付による日数差を求める
```
mysql> select to_days(now()) - to_days('2017-05-27 00:00:00');
+-------------------------------------------------+
| to_days(now()) - to_days('2017-05-27 00:00:00') |
+-------------------------------------------------+
|                                               3 |
+-------------------------------------------------+
```

日付の加算
```
mysql> select date_add(now(), interval 3 day);
+---------------------------------+
| date_add(now(), interval 3 day) |
+---------------------------------+
| 2017-06-02 08:24:44             |
+---------------------------------+
```

### CONCAT
CONCATを使うと文字列の結合ができるようになる。
```
mysql> SELECT CONCAT( 'http', '://', 'www.google.com', ':8080' );
+----------------------------------------------------+
| CONCAT( 'http', '://', 'www.google.com', ':8080' ) |
+----------------------------------------------------+
| http://www.google.com:8080                         |
+----------------------------------------------------+
```

どれか1つでもNULLが入ると結果はNULLとなる。
```
mysql> SELECT CONCAT( 'My', 'SQL', '関数', NULL );
+---------------------------------------+
| CONCAT( 'My', 'SQL', '関数', NULL )   |
+---------------------------------------+
| NULL                                  |
+---------------------------------------+
```


### 型変換

```
mysql> SELECT CAST(CONCAT('123', '456') AS SIGNED);
+--------------------------------------+
| CAST(CONCAT('123', '456') AS SIGNED) |
+--------------------------------------+
|                               123456 |
+--------------------------------------+

mysql> SELECT CAST(CONCAT('-123', '456') AS UNSIGNED);
+-----------------------------------------+
| CAST(CONCAT('-123', '456') AS UNSIGNED) |
+-----------------------------------------+
|                    18446744073709428160 |
+-----------------------------------------+

mysql> SELECT CAST(123 + 456 AS CHAR);
+-------------------------+
| CAST(123 + 456 AS CHAR) |
+-------------------------+
| 579                     |
+-------------------------+
```


### 数値計算関連
絶対値を取得する
```
mysql> select abs(-123);
+-----------+
| abs(-123) |
+-----------+
|       123 |
+-----------+
```

余りを表示する
```
mysql> select mod(123, 10);
+--------------+
| mod(123, 10) |
+--------------+
|            3 |
+--------------+
```

四捨五入する
```
mysql> select round(10.875, 2);
+------------------+
| round(10.875, 2) |
+------------------+
|            10.88 |
+------------------+
```


指定した桁数で切り詰める
```
mysql> select truncate(10.345 + .006, 2);
+----------------------------+
| truncate(10.345 + .006, 2) |
+----------------------------+
|                      10.35 |
+----------------------------+
```

指定した桁数で切り上げる
```
mysql> select truncate(10.345 + .9, 0);
+--------------------------+
| truncate(10.345 + .9, 0) |
+--------------------------+
|                       11 |
+--------------------------+
```

それ以上の最小の整数を返す
```
mysql> SELECT CEIL(10.345), CEIL(-10.345), TRUNCATE(10.345+.9, 0) t1, TRUNCATE(-10.345+.9, 0) t2;
+--------------+---------------+----+----+
| CEIL(10.345) | CEIL(-10.345) | t1 | t2 |
+--------------+---------------+----+----+
|           11 |           -10 | 11 | -9 |
+--------------+---------------+----+----+
```

平方根を返す
```
mysql> select sqrt(2.0);
+--------------------+
| sqrt(2.0)          |
+--------------------+
| 1.4142135623730951 |
+--------------------+
```

3桁ごとにカンマを付与する。第２引数は小数点以下桁数を表す。
```
mysql> SELECT FORMAT(1234567, 0);
+--------------------+
| FORMAT(1234567, 0) |
+--------------------+
| 1,234,567          |
+--------------------+

mysql> SELECT FORMAT(1234567, 2);
+--------------------+
| FORMAT(1234567, 2) |
+--------------------+
| 1,234,567.00       |
+--------------------+
```

文字を数値に変換する
```
mysql> select '0123';
+------+
| 0123 |
+------+
| 0123 |
+------+

mysql> select '0123' + 100;
+--------------+
| '0123' + 100 |
+--------------+
|          223 |
+--------------+
```

べき乗を取得する
```
mysql> SELECT pow(2, 10);
+------------+
| pow(2, 10) |
+------------+
|       1024 |
+------------+
```
