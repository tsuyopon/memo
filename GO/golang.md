# 詳細



### 変数初期化
i:=0で変数iを0に初期化します。
```
func main() {
    for i := 0; i < 10; i++ {
        fmt.Println("hello, world")
    }
}
```

局所変数に限り、次のように := を使って定義することができます。
```
変数名1, 変数名2, ..., 変数名n := 初期値1, 初期値2, ..., 初期値n
```

たとえば、次のように記述します。
```
a, b := 10, 20
c, d := 1.234, 5.678
```
