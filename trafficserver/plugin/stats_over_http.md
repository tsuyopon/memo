# 概要
HTTP統計プラグインはJSON形式で各種メトリックを取得するためのプラグインです。

# 使い方

_statsエントリポイントを変更したい場合には、plugin.configの設定に次のように引数を指定することで可能です。
```
stats_over_http.so 81c075bc0cca1435ea899ba4ad72766b
```

次のようにして、指定するパスを変更すること可能になります。
```
$ curl http://localhost/81c075bc0cca1435ea899ba4ad72766b
```


### CSV形式で取得する
Acceptにtext/csvを指定することでCSV形式の出力で応答を得ることが可能です。
```
$ curl -H "Accept: text/csv" http://localhost:8080/_stats
```
